Title:
문서 생성
Author
Seokhwan Kim
Date
Description:

문서의 생성이란, 크게 두 가지의 항목을 포함한다.

첫째, 특정 format 의 문서 생성이다. 예를 들어, SATES_DOC_EXAMPLE.txt 과 같은
텍스트 문서를 html, rtf, pdf 등으로 생성 가능해야 한다.

둘째, link 의 생성이다. 모든 문서는 반드시 연결되어야 한다.
사용자가 입력한 정보를 기반으로 hyperlink 형태의 link 가 생성되어야 한다.
