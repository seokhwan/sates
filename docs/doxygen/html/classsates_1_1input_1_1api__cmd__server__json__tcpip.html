TCP/IP 서버 open 의 준비를 한다.

Parameters
addr1Local PC 의 IP 주소
addr2서버를 open할 포트번호

일반적이라면 open 함수에서 client 의 accept 를 하지만, 여기서는 실행하지 않고 단순히 ip 및 port 번호만을 확보한다. 이는 run 함수가 async 로 동작하기 때문에 가능하다. 즉, 따라서 이 server 의 run 함수를 실행한 이후, client 사이드에서 접속이 가능하다.
