@title
테스트 실행

@author
Seokhwan Kim

@date

@desc
RU_0002_ADD_TESTCASE 에서 이야기한 것과 같이 테스트 케이스를 추가하면,
SW 프로젝트는 응당 이에 상응하는 테스트 코드가 작성되어야 한다.
그리고, 시스템은 이 테스트 코드를 실행할 수 있어야 한다. 

또한, 테스트 코드는 일반적으로 매우 긴 시간이 요구되는 테스트가 있을 수 있기 때문에
테스트 실행 시, 테스트 리스트를 편집할 수 있어야 한다. 즉, 사용자는 특정 테스트 
케이스만을 실행하거나 혹은 이번에 실행하지 않을 테스트 케이스를 지정할 수 있어야 한다.
